ABOUT

ในปัจจุบันการส่งจดหมายนั้นก็ยังเป็นที่นิยมอยู่ โดยเฉพาะเอกสารตอบรับต่างๆที่ส่งมาทางบ้าน แต่ในบางครั้งก็มีการลืมหรือ ไม่ทราบว่าได้มีจดหมายมาส่ง ดังนั้นจึงเกิดเป็นโครงงาน Mail alerts แจ้งเตือนจดหมายในตู้ไปรษณีย์ โครงงานชิ้นนี้จัดทำเพื่อช่วยเหลือในการแจ้งเตือนจดหมายที่มาส่งในกรณีที่เราไม่ได้อยู่บ้าน หรือกำลังทำธุระต่างๆ โดยเซ็นเซอร์จะทำหน้าที่ตรวจจับจดหมายที่มีการหย่อนลงมาในตู้จดหมาย และทำการส่งข้อความเตือนผ่านทาง Application Line เพื่อแจ้งให้ทราบแก่ผู้ใช้งาน ทำให้สามารถแก้ปัญหาการหลงลืมหรือการไม่ทราบต่อจดหมายที่ส่งมา

OBJECTIVE

  • สามารถแสดงรายละเอียดของวันและเวลาที่มาส่งในตู้จดหมายผ่านทาง Application Line
